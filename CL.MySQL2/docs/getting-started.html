📖 Overview

CL.MySQL2 is a fully integrated MySQL database library for the CodeLogic framework, providing a modern, type-safe approach to database operations with comprehensive support for CRUD operations, complex queries, and advanced features.

💡 Built for CodeLogic 2.0
This library is specifically designed to integrate seamlessly with the CodeLogic framework's logging, configuration, and dependency injection systems.

What Makes CL.MySQL2 Special?

  • Full Framework Integration: Works seamlessly with CodeLogic's logging, configuration, and DI
  • Type Safety: Compile-time checking with strong typing throughout
  • Fluent API: Readable, chainable query construction
  • Modern C#: Async/await, nullable reference types, latest language features
  • Performance: Connection pooling, caching, and optimized operations

✨ Features

🗄️ Repository Pattern

Generic repository with CRUD operations, caching, and automatic entity mapping.

🔍 QueryBuilder

Fluent API for building complex queries with joins, aggregations, and pagination.

🔌 Connection Management

Optimized pooling, health checks, and support for multiple databases.

📊 Type Conversion

Automatic conversion between C# and MySQL types including modern types.

📝 Comprehensive Logging

Integrated logging for all operations including slow query detection.

⚡ Performance

Caching, connection pooling, and optimized queries for high performance.

📦 Installation

1 Add Project Reference

Add CL.MySQL2 to your project file:

<ItemGroup>
  <ProjectReference Include="path/to/CL.MySQL2/CL.MySQL2.csproj" />
</ItemGroup>
2 Verify Dependencies

CL.MySQL2 requires:

  • CL.Core - Core CodeLogic abstractions
  • CodeLogic - Main framework
  • MySqlConnector - MySQL driver (automatically included)
3 Build Your Project 
dotnet build
✅ Installation Complete!
The library will be automatically discovered and loaded by the CodeLogic framework.

⚙️ Configuration

Basic Configuration

Create a mysql.json file in your config directory:

{
  "host": "localhost",
  "port": 3306,
  "database": "your_database",
  "username": "root",
  "password": "your_password",
  "min_pool_size": 5,
  "max_pool_size": 100,
  "connection_timeout": 30,
  "enable_logging": true,
  "enable_caching": true,
  "log_slow_queries": true,
  "slow_query_threshold_ms": 1000
}

Configuration Options

Option Type Default Description
host string localhost Database server hostname
port number 3306 Database server port
database string - Database name/schema
min_pool_size number 5 Minimum connections in pool
max_pool_size number 100 Maximum connections in pool
enable_caching boolean true Enable query result caching
slow_query_threshold_ms number 1000 Threshold for slow query logging
⚠️ Security Note
Never commit database passwords to version control. Use environment variables or secure configuration management.

🚀 Quick Start

Step 1: Get the Library Instance

// Get CL.MySQL2 from the library manager
var mysql2 = libraryManager.GetLibrary<MySQL2Library>();

Step 2: Create a Model

using CL.MySQL2.Models;

[Table(Name = "users")]
public class User
{
    [Column(DataType = DataType.Int, Primary = true, AutoIncrement = true)]
    public int Id { get; set; }

    [Column(DataType = DataType.VarChar, Size = 100, NotNull = true)]
    public string Username { get; set; }

    [Column(DataType = DataType.VarChar, Size = 255)]
    public string Email { get; set; }
}

Step 3: Perform Operations

// Get repository
var userRepo = mysql2.GetRepository<User>();

// Create
var newUser = new User
{
    Username = "alice",
    Email = "alice@example.com"
};
var result = await userRepo.InsertAsync(newUser);

// Read
var user = await userRepo.GetByIdAsync(1);

// Update
user.Data.Email = "newemail@example.com";
await userRepo.UpdateAsync(user.Data);

// Delete
await userRepo.DeleteAsync(1);
🎉 That's it!
You now have a fully functional database layer with type safety and comprehensive error handling.

🏗️ Defining Models

Table Attribute

Define table properties at the class level:

[Table(
    Name = "users",
    Engine = TableEngine.InnoDB,
    Charset = Charset.Utf8mb4,
    Comment = "User accounts table"
)]
public class User { /* ... */ }

Column Attributes

Define column properties on each property:

// Primary key with auto-increment
[Column(DataType = DataType.Int, Primary = true, AutoIncrement = true)]
public int Id { get; set; }

// Required string with unique constraint
[Column(DataType = DataType.VarChar, Size = 100, NotNull = true, Unique = true)]
public string Username { get; set; }

// Indexed column
[Column(DataType = DataType.VarChar, Size = 255, Index = true)]
public string Email { get; set; }

// Timestamp with default value
[Column(DataType = DataType.DateTime, DefaultValue = "CURRENT_TIMESTAMP")]
public DateTime CreatedAt { get; set; }

// Property ignored in database operations
[Ignore]
public string TempData { get; set; }

Foreign Keys

[Column(DataType = DataType.Int, NotNull = true)]
[ForeignKey(
    ReferenceTable = "users",
    ReferenceColumn = "id",
    OnDelete = ForeignKeyAction.Cascade
)]
public int UserId { get; set; }

Supported Data Types

  • Numeric: TinyInt, SmallInt, Int, BigInt, Float, Double, Decimal
  • Date/Time: DateTime, Date, Time, Timestamp, Year
  • String: Char, VarChar, Text, TinyText, MediumText, LongText
  • Binary: Binary, VarBinary, Blob, TinyBlob, MediumBlob, LongBlob
  • Other: Json, Uuid, Bool, Enum, Set

📚 Repository Pattern

The Repository pattern provides simple, type-safe CRUD operations.

Getting a Repository

var userRepo = mysql2.GetRepository<User>();

// Use a specific database connection
var analyticsRepo = mysql2.GetRepository<Event>("Analytics");

CRUD Operations

Create (Insert)

var newUser = new User
{
    Username = "bob",
    Email = "bob@example.com",
    Age = 30
};

var result = await userRepo.InsertAsync(newUser);

if (result.Success)
{
    Console.WriteLine($"Created user with ID: {newUser.Id}");
}

Read (Select)

// Get by ID
var user = await userRepo.GetByIdAsync(1);

// Get by any column
var byEmail = await userRepo.GetByColumnAsync("email", "bob@example.com");

// Get all (with caching)
var allUsers = await userRepo.GetAllAsync(cacheTtl: 300); // Cache for 5 minutes

Update

user.Data.Email = "newemail@example.com";
var updateResult = await userRepo.UpdateAsync(user.Data);

Console.WriteLine($"Updated {updateResult.RowsAffected} row(s)");

Delete

var deleteResult = await userRepo.DeleteAsync(1);

Console.WriteLine($"Deleted {deleteResult.Data} row(s)");

🔍 QueryBuilder

QueryBuilder provides a fluent API for constructing complex SQL queries.

Getting a QueryBuilder

var qb = mysql2.GetQueryBuilder<User>();

// Or get the factory
var factory = mysql2.GetQueryBuilder();
var userQuery = factory.For<User>();

Basic Queries

// Simple WHERE clause
var active = await qb
    .WhereEquals("status", "active")
    .ExecuteAsync();

// Multiple conditions
var filtered = await qb
    .Where("age", ">=", 18)
    .Where("city", "=", "New York")
    .OrderByDesc("created_at")
    .Limit(50)
    .ExecuteAsync();

// LIKE queries
var gmail = await qb
    .WhereLike("email", "%@gmail.com")
    .ExecuteAsync();

// IN queries
var cities = await qb
    .WhereIn("city", "NYC", "LA", "Chicago")
    .ExecuteAsync();

Pagination

var pagedResult = await qb
    .WhereEquals("status", "active")
    .OrderByDesc("created_at")
    .ExecutePagedAsync(pageNumber: 2, pageSize: 20);

if (pagedResult.Success)
{
    var paged = pagedResult.Data;
    Console.WriteLine($"Page {paged.PageNumber} of {paged.TotalPages}");
    Console.WriteLine($"Total: {paged.TotalItems} items");

    foreach (var user in paged.Items)
    {
        Console.WriteLine($"- {user.Username}");
    }
}

Aggregations

// Count
var count = await qb
    .WhereEquals("status", "active")
    .CountAsync();

// Group by with aggregates
var stats = await qb
    .GroupBy("city")
    .Count("*", "user_count")
    .Avg("age", "avg_age")
    .ExecuteAsync();

Joins

var ordersWithUsers = await qb
    .Select("users.*", "orders.total_amount")
    .LeftJoin("orders", "users.id = orders.user_id")
    .Where("users.status", "=", "active")
    .ExecuteAsync();

Debugging Queries

// View the SQL that will be executed
var query = qb
    .Where("age", ">", 18)
    .OrderBy("username");

Console.WriteLine(query.ToSql());
// Output: SELECT * FROM `users` WHERE `age` > @p0 ORDER BY `username` ASC

🎯 Advanced Usage

Multiple Databases

// Register a second database
var analyticsConfig = new DatabaseConfiguration
{
    ConnectionId = "Analytics",
    Host = "analytics-db.example.com",
    Database = "analytics",
    Username = "analytics_user",
    Password = "secure_password"
};

mysql2.RegisterDatabase("Analytics", analyticsConfig);

// Use the analytics database
var analyticsRepo = mysql2.GetRepository<Event>("Analytics");
var analyticsQb = mysql2.GetQueryBuilder<Event>("Analytics");

Health Monitoring

// Check database health
var health = await mysql2.HealthCheckAsync();

if (health.IsHealthy)
{
    Console.WriteLine($"✓ {health.Message}");
}
else
{
    Console.WriteLine($"✗ {health.Message}");
    // Take corrective action
}

Transactions

var connectionManager = mysql2.GetConnectionManager();

var result = await connectionManager.ExecuteWithTransactionAsync(
    async (connection, transaction) =>
{
    // Multiple operations in a transaction
    using var cmd1 = new MySqlCommand(sql1, connection, transaction);
    await cmd1.ExecuteNonQueryAsync();

    using var cmd2 = new MySqlCommand(sql2, connection, transaction);
    await cmd2.ExecuteNonQueryAsync();

    return true;
});
// Transaction is automatically committed or rolled back

💡 Code Examples

User Search with Dynamic Filters

public async Task<List<User>> SearchUsers(UserSearchCriteria criteria)
{
    var qb = mysql2.GetQueryBuilder<User>();

    // Conditionally add filters
    if (!string.IsNullOrEmpty(criteria.Username))
        qb.WhereLike("username", $"%{criteria.Username}%");

    if (criteria.MinAge.HasValue)
        qb.WhereGreaterThan("age", criteria.MinAge.Value);

    if (criteria.Cities?.Any() == true)
        qb.WhereIn("city", criteria.Cities.ToArray());

    var result = await qb.OrderBy("username").ExecuteAsync();
    return result.Data ?? new List<User>();
}

Dashboard Statistics

public async Task<DashboardStats> GetDashboardStats()
{
    var userQb = mysql2.GetQueryBuilder<User>();

    var totalUsers = await userQb.CountAsync();
    var activeUsers = await userQb
        .WhereEquals("status", "active")
        .CountAsync();

    var recentUsers = await userQb
        .WhereBetween("created_at",
            DateTime.Now.AddDays(-7),
            DateTime.Now)
        .CountAsync();

    return new DashboardStats
    {
        TotalUsers = totalUsers.Data,
        ActiveUsers = activeUsers.Data,
        NewThisWeek = recentUsers.Data
    };
}
